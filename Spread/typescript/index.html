Typescript

Iniciar o projeto com o comando

    npm init
    npm i parcel-bundle

Setar o script de start no arquivo package.json

    "scripts": {
    "start": "parcel index.html"
    },

Rodar o projeto com o comando

    npm start

Abrir no navegador o link

localhost:1234

Criar o arquivo .gitignore para ignorar as pastas .cache,dist e node_modules
